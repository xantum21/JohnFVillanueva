John Villanueva website package

Core pages:
- index.html - homepage
- about.html - About page
- work.html - Work & Education page
- projects.html - Projects page
- life.html - interactive Life page
- timeline.html - interactive full-life timeline
- contact.html - professional, social, and gaming contact hub
- 404.html - custom not-found page

Launch support:
- assets/ - optimized images, icons, social card, certificates, and public resume PDF
- sitemap.xml and robots.txt - search discovery
- site.webmanifest - site identity metadata
- _headers - optional security headers for compatible static hosts
- LAUNCH-NOTES.txt - deployment and technical notes
- PRIVACY-REVIEW.txt - what is public, softened, and intentionally omitted
- VALIDATION.json - local link and asset validation report

The public site intentionally omits a phone number, home address, date of birth, license number, work schedule, and exact current workplace. Social and gaming profiles remain linked because they were explicitly selected for public use.
